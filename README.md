# AI Voice Assistant

A local AI voice assistant built using Python.

## Features

- Voice Commands
- Spotify Control
- System Commands
- Question Answering
- Modular Command Structure

## Technologies

- Python
- SpeechRecognition
- PyAudio
- OpenAI API
- Spotify API
